<h4 id="list-comments">List Comments</h4>

- **Authentication:** Optional

```
GET /api/v1/posts/{number}/comments
```

**Parameters**

<table>
  <tr>
    <th width="80">Name</th>
    <th width="90">Type</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>
      <code>number</code>
    </td>
    <td>
      <code>number</code>
    </td>
    <td>**Required.** The number of the post to get comments from.</td>
  </tr>
</table>

**Example**

```
GET <baseURL>/api/v1/posts/47/comments
```

<h4 id="add-comment">Add a Comment</h4>

```
POST /api/v1/posts/{number}/comments
```

**Parameters**

<table>
  <tr>
    <th width="120">Name</th>
    <th width="90">Type</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>
      <code>number</code>
    </td>
    <td>
      <code>number</code>
    </td>
    <td>**Required.** The number of the post to add the comment.</td>
  </tr>
  <tr>
    <td>
      <code>content</code>
    </td>
    <td>
      <code>string</code>
    </td>
    <td>**Required.** The content of the comment.</td>
  </tr>
</table>

**Example**

```
POST <baseURL>/api/v1/posts/47/comments
---
{
  "content": "Agreed, we'll soon need to act on this"
}
```

**Response**

```
200 OK
---
{
  "id":5675
}
```

<h4 id="edit-comment">Edit a Comment</h4>

- **Authentication:** Required
- **Required Role:** Collaborator or Administrator.

```
PUT /api/v1/posts/{number}/comments/{id}
```

**Parameters**

<table>
  <tr>
    <th width="120">Name</th>
    <th width="90">Type</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>
      <code>number</code>
    </td>
    <td>
      <code>number</code>
    </td>
    <td>**Required.** The number of the post that has given comment.</td>
  </tr>
  <tr>
    <td>
      <code>id</code>
    </td>
    <td>
      <code>number</code>
    </td>
    <td>**Required.** The id of the comment to be edited.</td>
  </tr>
  <tr>
    <td>
      <code>content</code>
    </td>
    <td>
      <code>string</code>
    </td>
    <td>**Required.** The new content of the comment.</td>
  </tr>
</table>

**Example**

```
POST <baseURL>/api/v1/posts/47/comments/5675
---
{
  "content": "Agreed, we need to act on this as soon as possible"
}
```

**Response**

```
200 OK
---
{}
```
